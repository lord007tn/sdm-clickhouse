# v0.1.0 Deliverables

## Product Scope

1. Multi-connection ClickHouse workspace with saved profiles and password storage outside SQLite.
2. Schema explorer for databases, tables, columns, and table DDL inspection.
3. SQL workbench with multi-tab editing, execution, cancellation, paging, formatting, and explain flows.
4. Query history and reusable snippets persisted locally.
5. Guarded insert, update, delete, create, and drop actions.
6. Connection diagnostics, profile import/export, metadata backup/restore, and startup recovery notice handling.
7. ClickHouse observability overview with storage, engine mix, active query pressure, and hot-table insights.

## Repository Readiness

1. MIT license and public project policies.
2. Issue templates and pull request template.
3. GitHub Actions for CI, label sync, PR labeling, PR title linting, stale management, and release publishing.
4. Cross-platform Tauri packaging with updater artifact generation.

## Release Outputs

1. Linux `.deb`
2. Linux `.AppImage`
3. Windows `.msi`
4. Windows NSIS setup `.exe`
5. Windows portable ZIP
6. macOS `.dmg`
7. `latest.json` updater manifest
