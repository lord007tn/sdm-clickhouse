# v0.1.0 Deliverables

## Product Scope

1. Multi-connection ClickHouse management with secure credential storage.
2. Schema explorer (databases, tables, columns, DDL view).
3. SQL workspace with multi-tabs, duplication, reorder, formatting, explain, cancellation, timeout control.
4. Query history and snippets persisted locally.
5. Guarded insert/update/delete and basic DDL operations.
6. Connection diagnostics, profile import/export, metadata backup/restore.
7. Audit log and app log views.
