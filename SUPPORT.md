# Support

Use the channel that matches the type of help you need:

- Bugs: open a GitHub issue with reproduction steps, app version, OS, ClickHouse version when known, and relevant logs or screenshots.
- Feature requests: open a GitHub issue and describe the workflow problem, expected behavior, and any alternatives you considered.
- Usage questions: open a question issue with your environment and what you already tried.
- Security vulnerabilities: do not open a public issue. Follow [SECURITY.md](./SECURITY.md).

Maintainers and contributors are volunteers. Clear reproduction details and small examples make it much easier to respond.
